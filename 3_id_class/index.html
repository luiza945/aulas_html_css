ID E CLASS

ID

O id funciona como um cpf, assim como só posso ver um numero de cpf, elemento só pode ter umvalor de id, o cpf é unico, ou seja, ninguem pode ter um cpf igual ao meu. isso significa que o id é unico, entao nao é possivel repetir o mesmo duas vezes.

CLASS

O class funciona de maneira oposta ao id. diferente ao id em que voce so pode ter um valor, no class voce pode ter varios valores. tambem diferente do id em que não se pode repetir um valor, no class voce pode utilizar o mesmo valor varias vezes.
